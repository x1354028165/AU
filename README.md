# 澳洲储能电站项目 (aus-energy)

## 项目简介

澳洲储能电站项目 —— 通过峰谷套利实现盈利的储能电站系统。

### 商业模式
- **业主方**：出资建设电站，不参与运营
- **运维方**：租赁电站进行运维，每年向业主支付运维费
- **盈利模式**：运维方通过峰谷套利（低价买电、高价卖电）获利
